# Number Guessing Game (Python)

A simple command-line Number Guessing Game built using Python.  
The computer generates a random number, and the user tries to guess it within limited attempts.

## Features
- Random number generation
- User input-based guessing
- Hint system (Too High / Too Low)
- Limited attempts
- Beginner-friendly logic

## Concepts Used
- Variables
- Loops (while)
- Conditional statements (if-elif-else)
- Random module
- User input handling

## How the Game Works
1. The computer generates a random number within a given range.
2. The user enters a guess.
3. The program provides hints:
   - Too High
   - Too Low
4. The game continues until:
   - The user guesses correctly, or
   - The attempts are exhausted.
